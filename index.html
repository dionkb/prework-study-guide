Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element contains information on the page author, publishing year, etc.
  • The entire HTML code is used to build the structure and format of the webpage.

CSS

  • The CSS code is used to add specific styling to your webpage.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is used to give logic to a webpage.
